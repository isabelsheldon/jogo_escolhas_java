Sobre o projeto:
Jogo interativo desenvolvido em Java com execução via terminal. O jogador toma decisões que alteram o fluxo da história, levando a diferentes caminhos até o desfecho final. O projeto foi criado com foco em prática de lógica de programação e estrutura de decisão.

Tecnologias:
Foram utilizados Java e Scanner (entrada de dados via terminal).

Estrutura:
Fluxo principal, fluxos alternativos, validação de opções inválidas e reinínio ao final da história.
A navegação ocorre por meio de estruturas condicionais (if/else) e métodos separados para cada etapa da narrativa.
